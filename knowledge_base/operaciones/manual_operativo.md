# Solution Service AI

# Manual Operativo

**Versión:** 1.0

---

# Objetivo

Establecer las prácticas operativas utilizadas por Solution Service AI para la gestión de proyectos y servicios.

---

# Metodología

Los proyectos se gestionan utilizando Scrum.

---

# Ceremonias

- Sprint Planning
- Daily Meeting
- Sprint Review
- Retrospective

---

# Gestión de incidencias

Toda incidencia deberá registrarse en el sistema corporativo.

---

# Control de calidad

Cada entrega será validada antes de pasar a producción.

---

# Mejora continua

Al finalizar cada Sprint se realizará una retrospectiva para identificar oportunidades de mejora.

---

# Preguntas frecuentes

## ¿Qué metodología utiliza la empresa?

Scrum.

## ¿Cada cuánto se realiza la Daily?

Todos los días hábiles.

## ¿Quién registra una incidencia?

El responsable que detecte el problema.

## ¿Cuándo pasa un desarrollo a producción?

Después de superar el control de calidad.
